## 🎯 Key Projects

### 🔐 SOC Lab & Threat Detection
Built a comprehensive SOC environment simulating real-world attacks, ingesting logs into Splunk, and developing correlation logic. Scaled telemetry to AWS for cloud SOC visibility.

### 🤖 Automated Threat Detection System
Developed Python-based security tools using Pandas and Watchdog to detect suspicious domains and monitor unauthorized file access, integrated with Splunk SIEM via API.

### 🧠 ML-based Intrusion Detection
Created a TabNet-based network intrusion detection model using PyTorch and cuDNN, providing statistical insights into model decisions based on feature variations.

### 🕵️ Malware Analysis & Forensics
Analyzed malware using IDA Pro, Ghidra, and OllyDbg to uncover process hollowing, anti-debugging, and sandbox evasion techniques. Conducted insider threat simulations with SANS-standard forensic reporting.
